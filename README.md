# Network Service Port Scanner

## Overview
A lightweight network reconnaissance tool designed to identify open services on a target host. [span_2](start_span)This project demonstrates my understanding of **Networking Fundamentals**, including TCP/IP and handshake protocols[span_2](end_span).

## How it Works
The script utilizes the `socket` library to attempt connections on common ports (21, 22, 80, 443). [span_3](start_span)It provides immediate feedback on whether a service is "OPEN" or "CLOSED," mimicking tools like Nmap[span_3](end_span).
